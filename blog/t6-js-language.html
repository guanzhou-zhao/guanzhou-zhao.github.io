Describe parentheses () in JavaScript language.

() after function name is to revoke this function, () can be filled with parameters seperate by comma.

() after if, while keywords contains condition statements that will be treated as boolean type.

() after switch contains the parameter that will be the switch.

() gives statements higer priority to be executed.

Describe brackets [] in JavaScript language.

[] after an object or an array is to access a property or item of them.

var arrayName = []; or var arrayName[1, 2] is a way to declare an array.

Describe braces {} in JavaScript language.

{} surrounds a block of code. {} is put after flow control, loop, or function name.

Describe single quotes '', double quotes "" in JavaScript language.

'' and "" can both be used to declare a string.
