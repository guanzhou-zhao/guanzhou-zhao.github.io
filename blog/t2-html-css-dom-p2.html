What is grid based design?

Our eyes has looking pattern when we looking at things, which is looking from top left to top right, then from top to bottom. when eyes move down, they also follow pattern from left to right.

Grid based design utilise this pattern to design webpage that has grids from left to right, top to bottom, which is rows and columns. Within a column, there can be rows and columns inside too.

Whats all the hype about responsive webpages?

People are accessig websites using different devices which have different resolutions. The responsive webpages are not in fixed size and can response to different devices with different resolution to resize objects of webpage to make things look great.

What is semantic structure?

Semantic structure is using meaningful tags to contain corresponding content.

Paragraphy, headings, both can contain text, but it's better to use heading to contain heading. And the bigger heading contain the more inportant heading.

The process of building a webpage.

Using grids and semantic structure to layout design first, then refining the background, margin, padding, border, fonts. Code -> Test -> Refine.
