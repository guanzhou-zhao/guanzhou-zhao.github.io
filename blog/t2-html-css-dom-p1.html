Html Css DOM p1

Posted on July 7, 2016

  • Can you think of an analogy to describe HTML, CSS and the DOM? How would you describe it to your non-technical friend?

    Html is lke the structure of a building. CSS is decoration. DOM is the map of the building.

    Just like buildings can have different amount of rooms. Html can have any numbers of divisions. Rooms can have any kind of objects and divisions can have any kind of web objects too. Css determines the size, color, space of or between divisions. Decoration changs these attributes of a room. With DOM we can refer to any division in a webpage. With a map we can reach any space or objects in a building.

  • What is meant by boxifying design?

    Any objects in a webpage is a rectangular box. when implementing a design, firstly define the boxes in the design page.

  • What is the box model?

    Any object is a box. box has margin, border, padding and content.

    With the style attribute of "box-sizing: border-box", the "width, height" attribute define the size of the addition of width of border, padding and content.
