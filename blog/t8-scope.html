What 'scope' is, and how it works in JavaScript.

"Scope" is a range in which a variable is accessible. Global variable is with largest scope, which can be accessed anywhere. A variable is also avaible in subrange and can be covered by another variable with same name in sub-scope.

When a variable is accessed, system look for in local scope first, then to go parent scope, until to global scope. If nothing found, system error shows that the variable is undefined.
